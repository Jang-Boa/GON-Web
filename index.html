망막신경섬유층 결손(RNFL defect) 평가 실험
검사 시작 전 주의사항
본 실험에 참여해주셔서 감사합니다.

본 실험은 테스트 중 마우스를 활용하여
안저 영상에서 보여지는 RNFL Defect 영역의 상하 경계를 경계가 구분되는 곳까지 그려야 합니다.
    - “RNFL defect 무” 선택시 그리기 기능이 동작하지 않습니다.
    - “RNFL defect 유” 선택 후 영상에 직접 그리기가 가능합니다.
    - 클릭을 통해 line 그리기가 가능 하며 더블 클릭으로 해당 line 그리기를 종료 할 수 있습니다.
    - UNDO 버튼 혹은 crtl + z 로 영상에 그린 그림의 undo를 할 수 있습니다.
    - REDO 버튼 혹은 crtl + y 로 영상에 그린 그림의 redo를 할 수 있습니다.

케이스 별 진단 시간이 측정되며, RNFL Defect 영역을 그리는 툴이 익숙치 않을 것을 고려하여
실제로 실험을 진행하기에 앞서 우측 하단 "가이드 페이지 이동" 버튼을 눌러 연습을 진행하시길 바랍니다.

(가이드 페이지의 케이스는 정답과 무관하게 옵션 선택 기능 및 RNFL 영역 그리기 기능 연습을 시행해주시기 바랍니다.)
